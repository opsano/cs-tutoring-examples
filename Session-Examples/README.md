# Session Examples

This folder contains code written live during tutoring sessions. These examples are quick demonstrations created to explain concepts, fix bugs, or walk through problems step-by-step.

### About These Files

- They are not polished projects.
- They are meant to capture the exact ideas covered during a session.
- Each subfolder is typically labeled by topic or date.

Use these examples to revisit explanations discussed during tutoring and reinforce your understanding of the material.
