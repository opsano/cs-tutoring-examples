# CS Tutoring Examples

This repository contains example code, walkthroughs, and reference materials used during computer science tutoring sessions. All content is organized by programming language and topic to help students quickly find what they need for review or practice.

### Folder Overview

- **Java/** - Java examples for data structures, algorithms, recursion, and OOP.
- **C++/** - C++ examples covering syntax, pointers, classes, and data structures.
- **Python/** - Python examples for fundamentals, algorithms, and problem-solving.
- **Session-Examples/** - Code written live during tutoring sessions for quick reference.

These examples are designed to reinforce concepts discussed during sessions and provide students with clear, simple references.
