# Java Examples

This folder contains Java examples used in tutoring sessions. These files demonstrate core concepts from introductory and intermediate Java courses.

### Included Concepts

- Fundamentals (variables, loops, conditionals)
- Arrays and Strings
- Object-Oriented Programming (classes, inheritance, interfaces)
- Recursion
- Data Structures (linked lists, stacks, queues, etc.)
- Sorting and searching algorithms

Use these examples as a reference when reviewing class material or practicing Java coding skills.
